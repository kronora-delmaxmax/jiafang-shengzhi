# 《甲方模拟器·升职记》——文字互动游戏剧情脚本 v1.0

> **类型**：职场剧情 / 文字互动 / 升职养成
> **平台**：iOS / Android（原生或 H5）
> **题材来源**：小红书、微博、知乎公开职场故事（乙方学/乙里乙气/升职/站队/改稿/饭局）
> **目标用户**：18-35 岁，体验过或围观过职场的玩家
> **单局时长**：90-150 分钟（10-15 幕主剧情 + 20+ 抉择）
> **核心卖点**：「你做的每一个选择，都在公司里流传」——朋友圈 + 口碑系统

---

## 一、一句话定位

> **你是一个被空降到广告公司的小主管，要在 6 个月内升到合伙人。每一次会议、饭局、群聊、朋友圈都是生死局。**

---

## 二、核心机制设计

### 2.1 数值体系（5 条属性线）

| 属性 | 简写 | 范围 | 效果 |
|------|------|------|------|
| **业务力** | 业务 | 0-100 | 提案通过率 / 客户满意度 |
| **人情力** | 人情 | 0-100 | 同事 / 老板信任度 |
| **话术力** | 话术 | 0-100 | 谈判 / 挽留客户成功率 |
| **抗压值** | 抗压 | 0-200 | 累积压力 ≤100 状态正常；>100 触发崩溃事件；=200 game over |
| **口碑值** | 口碑 | 0-100 | 决定最终结局走向 |

**初始值**：业务 30 / 人情 30 / 话术 30 / 抗压 0 / 口碑 50

### 2.2 道具 / 卡片

| 道具 | 描述 | 一次性的？ |
|------|------|------|
| 🪪「工牌」 | 你的职级，决定你能接的项目大小 | 长期 |
| ☕「半糖咖啡」 | 抗压 -10 | 一次性 |
| 🕯️「蜡烛」 | 加班到凌晨的痕迹，触发同事共鸣 | 一次性 |
| 📱「老板微信置顶」 | 关键时刻可发私聊 | 长期 |
| 🌹「玫瑰花 emoji」 | 乙方学大师专属，可在群聊用 | 一次性 |
| 🍱「加班外卖」 | 团队抗压 -5，忠诚 +1 | 一次性 |
| 📇「客户名片」 | 你的客户资源 | 长期 |
| 📕「行业黑皮书」 | 业务 +10 临时 | 一次性 |

### 2.3 朋友圈动态系统（核心爽点）

**机制**：玩家每次做出重要选择，会触发一条「朋友圈动态」，展示给玩家：
- 当事人头像（玩家 + 配角）
- 动态文案（带 emoji、句号、波浪号）
- 点赞 / 评论 / 互动
- **不同选择会生成不同动态**——这是「你的玩法你的故事」的核心

**示例**（选择 A：硬刚甲方的后果）：
> **王铁锤**（你的老板）
> 今天把一个拖款 3 个月的客户怼回去了。不破不立。
>
> 👍 李总 财务总监：我敬你
> 👏 小张 AE：老大威武
> 💬 张设计：太爽了，希望全行业推广
> 💬 王司机：别冲动啊铁锤
> 3 分钟前

**示例**（选择 B：跪舔甲方的后果）：
> **王铁锤**
> 还在跟客户通电话。一句"我们尽力"说了二十遍。
>
> 👍 李总：辛苦了
> 👏 小张 AE：老大扛住
> 💬 王司机：注意身体
> 1 小时前

**反差机制**：玩家能通过朋友圈看到自己做的选择**在别人眼里什么样**——是威风、是憋屈、是圆滑、是无能，全部具象化。

### 2.4 抗压值与崩溃事件

| 抗压值 | 状态 |
|--------|------|
| 0-50 | 🟢 正常 |
| 51-100 | 🟡 疲惫（部分对话灰显，需要选择"撑住"才能继续）|
| 101-150 | 🟠 边缘（出现幻觉对话、心理独白） |
| 151-199 | 🔴 崩溃边缘（同事开始关心你、老板找你谈话） |
| 200 | 💀 **Game Over**：「你选择了挂断电话，从此再也没接到响铃」 |

### 【第 1 幕】入职第一天
**日期**：周一 9:00
**地点**：公司大堂
**抗压**：0

**剧情开场**：
> 你站在写字楼下，深吸一口气。33 岁的林舟，前大厂 P7，离职 3 个月后，HR 把你塞进这家 30 人不到的小广告公司当小组长。
>
> 工资降了 30%。但你告诉自己：这是最后一次机会。
>
> 电梯里，你盯着镜子里自己的脸。手机震了一下。

**抉择 1.1**：电梯里有人，是前同事赵欣，她也入职了。**怎么办？**

| 选项 | 效果 |
|------|------|
| A. 主动打招呼 "好巧！" | 人情 +2，赵欣好感 +1 |
| B. 假装看手机不认 | 人情 -1，赵欣警惕 |
| C. 主动示弱 "我其实挺紧张的" | 人情 +1，话术 -1 |

**第一回合办公室场景**：
- 老板王铁锤在黑板上写"本月目标 100 万"
- 张设计埋着头改图，不抬眼
- 小张 AE 端着咖啡问："林哥，要不要加糖？"
- 群消息"热烈欢迎林舟加入"——只有李总没点赞

### 【第 2 幕】第一次提案——双鱼项目
**日期**：入职第 3 天
**地点**：双鱼科技会议室
**抗压**：15

**剧情**：
> 张总坐在主位，翘着二郎腿。
> "听说你们换将了？让新人来讲，行。我时间宝贵，10 分钟。"
>
> 你看着 PPT——这份 PPT 是张设计熬了两个通宵做的，但 5 页里有 3 页的图已经被张总之前的助理退回过一次。

**抉择 2.1**：开场前 30 秒，你发现 PPT 第 7 页的"产品图"是个 placeholder 空图。**怎么办？**

| 选项 | 效果 |
|------|------|
| A. 直接讲，跳过这一页 | 业务 -5，张总皱眉 |
| B. 承认失误，立刻现场说"这页我们重点谈策略" | 话术 +3，业务 +1 |
| C. 偷偷给小张 AE 发消息让他立刻顶上 | 人情 +2，团队好感，但风险大 |

**抉择 2.2**：张总听完提案说："还行吧。整体方向我同意，但 logo 要五彩斑斓的黑，能发光的。下周给我 3 版。"**怎么回？**

| 选项 | 效果 |
|------|------|
| A. "好的好的，没问题！马上安排~ 辛苦张总啦🌹" | 业务 +2，张总好感 -1（觉得你没原则） |
| B. "张总，logo 的色彩我们先出 2 版方向对一下，再展开 3 版细节，避免返工，您看行吗？" | 话术 +5，业务 +3，**张总愣了一秒**（第一次有人顶他）|
| C. "五彩斑斓的黑……这个，张总您能具体说说想用在哪个场景吗？我理解下来可能有几个不同的处理方式。" | 话术 +8，**触发张总沉默 10 秒**，之后开始认真讨论 |

**朋友圈动态触发**：根据选择不同，会生成不同朋友圈动态。

### 【第 3 幕】第一次被逼疯
**日期**：入职第 8 天，凌晨 1:23
**地点**：你的工位
**抗压**：+25

**剧情**：
> 群里张总又发消息：
> "方案不错，但整体感觉不对。我们再聊聊？"
> 1 分钟前
> "你明天上午有空吗？想跟你通个电话"
> 5 秒前
>
> 你盯着屏幕。8 天前你提的方案已经改了 11 版。
> 现在他要你"再聊聊"。
> 张设计在你工位后面的沙发上缩着，披着羽绒服，眼里没有光。

**抉择 3.1**：**你怎么回？**

| 选项 | 效果 |
|------|------|
| A. "好的张总，我马上看消息哈~ 您方便的时候随时打来~ 🌹" | 抗压 +15，业务 -3，人情 -2（张设计看你的眼神变了）|
| B. "张总，已经第 11 版了。这次咱们能不能当面或者视频，把方向一次定下来，避免再返工？辛苦您了。" | 话术 +6，**张总沉默 5 分钟后回："行，明早 10 点视频。"** |
| C. 已读不回，明天再说 | 抗压 +5，业务 -8，**薛琳（助理）私下找你："他明天要拿这个去见 CEO，你再不放点东西，他要换乙方"** |
| D. 截图发到朋友圈： "11 改了 11 版。凌晨 1:23。" | 口碑 +5（同行共鸣），**但张总第二天会看到**——后果随机 |

**抉择 3.2**：第二天下午 4 点，你终于交付了"最终版"。5 点，张总说："嗯……还是用第一版吧。"

**你的反应是？**

| 选项 | 效果 |
|------|------|
| A. 默默关掉电脑 | 抗压 +20，团队士气 -5 |
| B. 给张设计发消息："哥，11 改完用第一版。我请你喝咖啡。" | 抗压 +10，人情 +5，张设计给你一包烟（暗示认可）|
| C. 在群里发："好的张总，这就给您回第一版~" | 业务 -5，口碑 -3（同行知道后骂你） |

### 【第 4 幕】饭局站队
**日期**：入职第 18 天，晚上 7:00
**地点**：某日料店包间
**抗压**：50（临界）

**剧情**：
> 王铁锤叫你去陪客户李老板（不是李总，是新潜在客户）吃饭。
> "林舟，今天的局你来主导。我跟李老板喝两杯就走，剩下的交给你。"
> 饭局上，李老板开始吹牛，张设计低头猛吃，小张 AE 在旁边端茶递水。
> 突然李老板说："小王啊，你们公司那个赵欣，我听说能力不错，下周让她来跟我对接。"

**抉择 4.1**：**怎么接？**

| 选项 | 效果 |
|------|------|
| A. "李老板好眼光，赵欣确实是我们这的优秀同事。不过今天是我来对接，事后我可以做全套内部协调。" | 话术 +8，**李老板放下筷子看了你 3 秒** |
| B. "没问题李老板，我回去跟王总说一声，赵欣下周一定到位。" | 抗压 +5，但**回公司后王铁锤找你谈话："谁让你现场答应的？"** |
| C. "李老板，今天是我主导这个局，是不是先让我这边把项目聊完，您看后续哪个同事更合适？毕竟合适最重要。" | 话术 +10，人情 -3（赵欣记恨），**李老板开始认真听你聊项目** |

**抉择 4.2**：饭局结束，李老板让你跟他单独去酒吧"再聊两句"。王铁锤还没走，看着你。**怎么办？**

| 选项 | 效果 |
|------|------|
| A. 跟李老板去 | 抗压 +10，话术 +5，**第二天的朋友圈会出现你和李老板的合照**（同行骂你"巴结"）|
| B. 礼貌拒绝，说家里有事 | 人情 +3，话术 -2，**李老板意味深长地笑**——他不会再来找你了 |
| C. 当着王铁锤的面婉拒："李老板，今天聊得挺好的，下次我做东，让我老板先送您？" | 话术 +12，人情 +5，**王铁锤第二天开会点名夸你** |

### 【第 5 幕】大客户流失危机
**日期**：入职第 1 个月零 5 天
**地点**：会议室
**抗压**：65

**剧情**：
> 张总（双鱼科技）打电话给你，声音发冷：
> "林舟啊，这个月你们的服务我不太满意。下个季度的续约，我考虑考虑。"
> 还没等你回，他就挂了。
>
> 你去查合同——这个客户占公司营收的 30%。
> 你知道，王铁锤还不知道这件事。

**抉择 5.1**：**告诉老板吗？怎么告诉？**

| 选项 | 效果 |
|------|------|
| A. 立刻当面汇报："王总，张总那边可能不续约了，我们现在需要 ABC 三个方案。" | 业务 +3，人情 +8，**王铁锤会跟你一起扛** |
| B. 等 3 天，自己先想办法解决再汇报 | 抗压 +20，**如果没解决，3 天后他发现会更糟**——口碑 -10 |
| C. 私下找张设计的意见后再汇报 | 人情 +5，业务 +2（获得张设计的情报：张总正在接触另一家乙方）|
| D. 转发给赵欣，让她一起背锅 | 口碑 -20，**赵欣反过来告你一状** |

**抉择 5.2**：你打听到张总接触的那家乙方报价比你低 20%。**怎么反扑？**

| 选项 | 效果 |
|------|------|
| A. 跟张总硬刚："张总，我们比他们更懂您的业务，您不能只看价格。" | 话术 +3，**但张总已经做了决定**——业务 -10 |
| B. 找到张总助理薛琳，让她在张总耳边吹风 | 业务 +5，**但要花 1 万块红包**（触发"是否动私账"抉择）|
| C. 用王铁锤的资源，请张总和他共同的朋友吃饭，从中撮合 | 业务 +10，人情 +3，**但消耗王铁锤的私人关系**（后续王铁锤对你有意见）|
| D. 直接让老板王铁锤出面，告诉他真相 | 业务 +5，**王铁锤亲自出马**——触发隐藏剧情：王铁锤认识张总的 CEO，20 分钟解决 |

### 【第 6 幕】刘老板出没
**日期**：入职第 3 个月
**地点**：行业大会现场
**抗压**：40

**剧情**：
> 你在一个行业大会上，遇到了行业大佬刘老板。
> 他端着一杯威士忌，跟你聊了 15 分钟。
> "小林啊，你这个人有点意思。哪天来我办公室坐坐？"
> 旁边的人听到这句话，看你的眼神都变了。
> 你知道——刘老板手里有 5 个亿的市场预算。

**抉择 6.1**：**怎么接？**

| 选项 | 效果 |
|------|------|
| A. "刘总过奖了，改天一定登门拜访。" 保持距离 | 业务 +2，**刘老板会忘**——这个客户拿不到 |
| B. "刘总，明天方便吗？我正好带了我们最新的方案。" | 话术 +10，**触发隐藏剧情：刘老板第二天主动约你** |
| C. 当场掏出手机加微信，把你团队最近做的案例发给他 | 话术 +15，**但小张 AE 看到这个动作，脸色变了**——公司里开始传你要跳槽 |

**抉择 6.2**：你跟刘老板深聊了 1 小时后，回家路上接到王铁锤的电话：
> "林舟，听说你今天在大会上加了刘老板微信？"
> 声音平静，但你知道他在意。

**怎么回？**

| 选项 | 效果 |
|------|------|
| A. 立刻汇报："王总，今天正好碰上刘老板，他很认可我们。我打算让他下周来公司参观，您要不要一起？" | 业务 +5，人情 +15，**王铁锤当天就带你见他的核心团队** |
| B. 实话实说："王总，刘老板想挖我。我没答应。我先把这件事告诉您。" | 人情 +20，**但你之后在公司会很尴尬**——他开始防着你 |
| C. 隐瞒不报 | 抗压 +15，**纸包不住火**——2 周后被赵欣截胡 |

### 【第 7 幕】升职答辩
**日期**：入职第 6 个月
**地点**：公司大会议室
**抗压**：80
**业务 + 人情 + 话术 + 口碑** —— 总和决定答辩顺序

**剧情**：
> 王铁锤召开合伙人会议。
> "林舟、赵欣，这次的合伙人预备人选，你们两个。"
> 赵欣冲你微笑。

## 六、支线剧情库（30+ 个小事件）

每幕之间穿插 1-2 个支线，每个支线会小幅影响属性+触发不同的朋友圈动态。

### 支线 1：凌晨 3 点的奶茶

## 七、乙里乙气等级系统（隐藏）

| 等级 | 称号 | 达成条件 |
|------|------|----------|
| L0 | 直肠子 | 从不乙里乙气 |
| L1 | 学徒 | 偶尔用"好的哈" |
| L2 | 入门 | 经常用"好的好嘞" |
| L3 | 熟练 | 熟练使用 emoji 表情包 |
| L4 | 进阶 | 开始说"咱们这边"而不是"我这边" |
| L5 | 大师 | "亲 这边建议先定个方向哈" |
| L6 | 宗师 | 一句话能叠 3 个波浪号 + 2 个 emoji |
| L7 | 太乙真人 | 朋友圈全是"辛苦您啦~🌹" |

**作用**：影响客户挽留率（高乙里乙气 → 客户粘性高 +10%），但**影响公司同事对你的看法**（"这人到底有没有立场"）。

## 九、技术实现要点（脚本层面）

### 9.1 核心数据结构

```json
{
  "player": {
    "name": "林舟",
    "business": 30,
    "human": 30,
    "speech": 30,
    "stress": 0,
    "reputation": 50,
    "yibei_level": 0
  },
  "npc_states": {
    "张总": { "trust": 0, "annoy": 0, "private_balance": 0 },
    "王铁锤": { "trust": 0, "suspicion": 0 },
    "张设计": { "trust": 0, "tired": 0 },
    "赵欣": { "trust": 0, "hostile": 0 }
  },
  "inventory": [],
  "wechat_moments": []
}
```

### 9.2 关键算法

1. **抉择影响计算**：每个选项有基础属性影响 + 隐藏 tag
2. **乙里乙气值**：A 选项（跪舔类）+1 Lv，B 选项（中性）+0，C 选项（硬刚）-0.5 Lv
3. **朋友圈触发**：每个重大选择后自动生成 1-2 条朋友圈
4. **结局分支判定**：第 7 幕答辩前，根据 5 大属性 + 关键 tag 触发对应结局

### 9.3 复玩性设计

- 8 个结局，每个都有"如果当初……"的暗示
- 第 2 次玩解锁"反派视角"——你可以扮演赵欣
- 第 3 次玩解锁"老板视角"——扮演王铁锤，看你作为下属做的所有事
- **彩蛋**：在某条隐藏朋友圈下面输入特定关键词，解锁"彩蛋结局"
