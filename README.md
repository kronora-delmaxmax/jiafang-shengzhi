# 甲方模拟器·升职记

> 你是一个被空降到广告公司的小主管，6 个月后要么升为合伙人，要么被优化。
> 每一次会议、饭局、群聊、朋友圈都是生死局。

## 🎮 立即玩

👉 **https://kronora-delmaxmax.github.io/jiafang-shengzhi/**

## ✨ 特性

- 🎬 **7 幕主线** + **10 个支线**（25-30 分钟一次跑通）
- 🎯 **17 个决策点**（每个选择都影响 5 大属性 + 7 个 NPC 信任度 + 8 级乙里乙气）
- 🏆 **7 个结局**（合伙人升职 / 乙学大师 / 创业线 / 带队跳槽 / 撑不住 / 被优化 / 留任主管）
- 💬 **朋友圈动态系统**（每个关键选择生成朋友圈）
- 🎓 **乙里乙气等级**（L0-L8，致敬网络梗）
- 📱 **微信式沉浸 UI**（滑卡决策 + 通知横幅 + 粒子特效）

## 🎭 4 个出身

- **林舟**（33 岁空降小组长）——前大厂 P7，失业 3 个月
- **张设计**（30 岁改稿狂魔）——Ctrl+S 是肌肉记忆
- **老周**（35 岁危机程序员）——最后一天到岗
- **小琪**（32 岁乙学练习生）——曾经 HRBP，现在乙方

## 🛠️ 技术

- 单文件 `index.html`（93KB）
- 纯 Vue 3 + 纯 CSS3
- 零构建，零依赖（仅 Vue 3 CDN）
- 兼容 iOS Safari / Android Chrome

## 🎮 操作

| 操作 | 功能 |
|------|------|
| 左滑 / 点击左选项 | 拒绝型选择（红）|
| 右滑 / 点击右选项 | 接受型选择（蓝）|
| ← / → 方向键 | 同上 |
| 顶部 📷 按钮 | 打开朋友圈 |
| 顶部 📊 按钮 | 显示/隐藏属性仪表盘 |
| 顶部 ‹ 重开 | 重置游戏（需确认）|
