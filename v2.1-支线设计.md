# v2.1 支线插入设计

> **目标**：把 v1.1 增量的 20+ 支线，精选 10 个适配 v2.0
> **总节点数**：23 → 33（+10）
> **总决策数**：7 → 12（+5，新增 5 个支线内决策）
> **单次跑通时长**：15 分钟 → 25-30 分钟

---

## 选支线原则

✅ **必须有真实素材支撑**（来自小红书/微博/知乎公开案例）
✅ **必须能改变数值**（压力/状态/NPC 信任度）
✅ **必须能触发朋友圈**（玩家做了选择有视觉反馈）
✅ **必须 1-3 分钟能跑完**（支线不能太长）

## 实现策略

### 1. 简单插入（最优先）
在 `await ctx.goto('act1_office')` 后插入：
```js
await ctx.goto('act1_office');
// 可选支线
const sideChoice = await ctx.decision(
  '接下来几天发生了一些事，你想——',
  '都经历一遍（玩 25 分钟）',
  '跳过支线（玩 15 分钟）'
);
// 实际是过场，不影响数值
```

但**这样不行**——支线必须**有真实选择和数值变化**。

### 2. 强制支线（推荐）
直接在 `act1_office` 末尾**拼接**支线节点代码：
```js
// 在 act1_office 末尾追加：
await ctx.wait(2000);
ctx.addInfo('⏰ 第二天 上午 10:15');
// ... 支线代码 ...
await ctx.goto('act1_choose_zhaoxin');
```

### 3. 概率触发（高级）
50% 概率插入支线：
```js
if (Math.random() < 0.5) {
  await ctx.goto('act1_side_zhao');
} else {
  await ctx.goto('act1_choose_zhaoxin');
}
```

**v2.1 用方案 2**（强制插入，确保每次都体验到支线）。

### 支线 1：副职的试探（act1_side_zhao）

**插入位置**：act1_office 末尾
**剧情**：你在茶水间遇到副职老张（不是张总，是公司的"前朝老臣"），他试探你对公司几个老员工的看法。
**真实素材**："前同事赵欣"原型 + "副职架空"经典

```javascript
act1_side_zhao: {
  async run(ctx) {
    ctx.setTime('10:15');
    ctx.addInfo('⏰ 第二天 上午 10:15 · 茶水间');
    await ctx.wait(800);
    await ctx.switchNpc('王铁锤');
    await ctx.sendFrom('王铁锤', '小舟', '10:15');
    await ctx.sendFrom('王铁锤', '过来一下', '10:15');
    await ctx.sendFrom('王铁锤', '我介绍个老员工给你认识', '10:15');
    await ctx.wait(1500);
    await ctx.switchNpc('王铁锤');
    await ctx.sendFrom('王铁锤', '老张', '10:16');
    await ctx.sendFrom('王铁锤', '这是新来的林舟', '10:16');
    await ctx.sendFrom('王铁锤', '老张在公司 8 年了', '10:16');
    await ctx.sendFrom('王铁锤', '你们聊聊', '10:16');
    await ctx.wait(2000);
    ctx.addNarrator('老张端着茶杯，微笑着看你。\n"小林啊，来公司多久了？"\n"业务熟了吧？"\n"那个新项目，听说张总挺重视的？"\n\n他在套你的话。\n你怎么回？');
    await ctx.wait(1500);
    const choice = await ctx.decision(
      '老张：\n"小林啊，来公司多久了？业务熟了吧？"\n他在套你的话。\n\n你怎么回？',
      '实话实说：刚来 3 天，还在熟悉\n(老张信任 +5，人情 +3)',
      '反向试探：张哥您看呢？\n(老张信任 -3，话术 +5)',
      '模糊应付：还行还行，还在学习哈~🌹\n(乙里乙气 +2)'
    );
    if (choice === 'left') {
      ctx.modify({ human: 3, 'npc.王铁锤.trust': 5, stress: 3 });
      await ctx.switchNpc('王铁锤');
      await ctx.sendFrom('王铁锤', '这小伙子实在', '10:18');
      await ctx.sendFrom('王铁锤', '我看好他', '10:18');
      ctx.addNarrator('老张笑了笑，没再问。\n但他记下了——这个新人不张扬。\n\n（王铁锤在远处听到你们的对话。）');
    } else if (choice === 'right') {
      ctx.modify({ speech: 5, 'npc.王铁锤.trust': -3 });
      ctx.addNarrator('老张脸一沉。\n"我没事问你干嘛？"\n\n（王铁锤在远处听到了你"老油条"的一面。）');
    } else {
      ctx.modify({ yibeiLevel: 2 });
      await ctx.switchNpc('王铁锤');
      await ctx.sendFrom('王铁锤', '……', '10:18');
      ctx.addNarrator('老张觉得你"会做人"。\n但他不知道你在想什么。\n\n（你的乙里乙气等级 +1。）');
    }
    await ctx.wait(2000);
    ctx.addMoment({
      author: ctx.state.name,
      text: '副职老张试探我。\n我没露底。\n他端着茶杯。\n我也端着。\n\n职场第一课：\n会说话比会做事重要。',
      time: '10:20',
      emoji: '🍵',
      likes: 1,
      comments: [{ name: '小张AE', text: '舟哥你真稳' }]
    });
    await ctx.wait(2000);
    await ctx.goto('act1_choose_zhaoxin');
  }
}
```
